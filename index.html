About The Developers

All the code required for NaTour can be found on our Github page! Below, you will find a detailed description of each members contributions.

Andrew D.

Google Vision, Web Dev

Used the Google Cloud Vision API for deciphering animals in the picture. Website was completed using HTML, CSS and JavaScript. Helped debug/program map and markers in React.

Tommy W.

UI, Services

Used Expo (React Native) to complete the User Interface of NaTour. Also used React Native to complete various services in NaTour such as the map, markers and camera.

Sophia T.

Firebase, React Native

Used Expo and React Native to create the information page opened up after Google Vision is completed. Also set up the firebase used to keep track of animals locations on the map.

Graeme W.

Wolfram API, React Native

Used the Wolfram API to retrieve information about animals retrieved from Google Vision. Used React Native to help implement the detailed information about the animal in the app.
